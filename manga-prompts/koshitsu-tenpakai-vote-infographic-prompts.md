# 皇室典範改正 — 投票ボタン用 個別論点画像プロンプト

共通スタイル: `manga-prompts/infographic-style-guide.md` の **7. 個別論点画像テンプレ** を使用する。

用途: 皇室典範改正ページの第1問「一番気になる論点」投票ボタン画像。

基準テイスト: 副首都ページの `docs/images/fukushuto-infographic-*.webp` と同じ、明るい civic-tech インフォグラフィック。白〜淡い水色背景、濃紺・鮮やかな青の見出し、角丸カード、柔らかい影、ドット・スパークル・グラフ風アクセントを使う。実在の皇族・政治家・政党ロゴ・政府エンブレムは描かない。

## 保存先

生成後 WebP に変換、各ファイルは `900x900` 目安、100KB以下を目標にする。

```
docs/images/
├── koshitsu-vote-kouseki.webp
├── koshitsu-vote-yoshi.webp

```

## 1. 女性皇族の皇籍維持

**ファイル名:** `koshitsu-vote-kouseki.webp`

```text
Create a polished 1:1 square Japanese infographic.

Theme:
皇室典範改正 の個別論点「女性皇族の皇籍維持」

Main title:
「女性皇族の皇籍維持」
「結婚後も皇族に残る？」

Core message:
女性皇族が結婚後も皇族として活動できる制度。皇族数の確保につながる一方、配偶者・子の扱いと皇位継承権の線引きが論点になる。

Composition:
Center:
簡略化した家系図。実在人物の顔ではなく中立的なシルエットを使う。女性皇族のシルエットが「結婚」の矢印を通っても、淡い青の枠「皇族」の中に残る構図。配偶者・子のアイコンには小さな「？」を付ける。

Around the center, place 4 rounded check cards:
1. 「結婚後も活動」 「公務の担い手を確保」
2. 「配偶者」 「皇族になるのか」
3. 「子ども」 「次世代まで残るのか」
4. 「継承権」 「皇位継承とは別論点」

Bottom conclusion band:
「皇族数の確保と制度の線引きが焦点」

Use the common “SNS反応まっぷ civic-tech infographic style”.
```

Create a polished 1:1 square Japanese infographic.

Center:

2. 「情報不足」 「制度の理解が必要」
3. 「賛否保留」 「判断を急がない」
4. 「複合論点」 「理由を分けて考える」
5. 「まず整理」 「気になる点から見る」

「わからない時は、論点を分けて見る」
