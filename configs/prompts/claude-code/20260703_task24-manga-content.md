## ワークフロー（テーマごとに繰り返す）

### Step 1: 漫画データ作成（自動）

スタンスマップの象限分布と代表投稿をもとにキャラクターを設計し、`configs/<topic>-reaction-map.json` の `manga` フィールドを追記・更新する。

**象限→キャラ設計の原則:**
- スタンスマップで最多の象限 → キャラAの立場・感情・台詞の雰囲気
- スタンスマップで第2位の象限 → キャラBの立場・感情・台詞の雰囲気
- 感情強度 e≥1.5 の代表投稿 → キャラの怒りや切実さの台詞ヒント
- 冷静 e≤0 の代表投稿 → ページ3（余韻）の論点や問いかけのヒント

**ページ構成（3ページ固定）:**
- ページ1: 導入 — どちらかのキャラの日常と問題への気づき
- ページ2: 対立 — 2キャラが出会い、価値観がぶつかる
- ページ3: 余韻・オープンエンド — 解決しない。読者に「あなたはどう思う？」と問う

### Step 2: GPTimage2プロンプト出力（自動）

以下の2ファイルを作成（`ai-copyright-prompts.md` の形式を踏襲）:

- `manga-prompts/<topic>-prompts.md`: キャラシート2枚 + 本番ページ3枚（計5枚）の英語プロンプト
- `manga-prompts/<topic>-vote-prompts.md`: 投票ボタン画像4枚の512×512英語プロンプト

**日本語フキダシについて:** GPTimage2は日本語テキスト描画が得意。ページプロンプトには `A speech bubble containing the Japanese text "台詞"` の形式で台詞を含める。

### Step 3: 画像生成（手動 — ユーザー作業）

このステップはクロード側では実行しない。プロンプトファイルを作成したら、ユーザーにGPTimage2での生成を依頼する。

配置先（ユーザーが配置）:
```
docs/images/
├── <topic>-manga-charsheet-1.webp   # キャラA
├── <topic>-manga-charsheet-2.webp   # キャラB
├── <topic>-manga-page-1.webp        # 導入ページ
├── <topic>-manga-page-2.webp        # 対立ページ
├── <topic>-manga-page-3.webp        # 余韻ページ
├── <topic>-vote-1-<label>.webp      # 投票ボタン画像1
├── <topic>-vote-2-<label>.webp      # 投票ボタン画像2
├── <topic>-vote-3-<label>.webp      # 投票ボタン画像3
└── <topic>-vote-4-<label>.webp      # 投票ボタン画像4
```

### Step 4: HTML更新（自動）

`docs/<topic>-reaction-map.html` に以下を追加（`ai-copyright-reaction-map.html` の実装を完全に踏襲）:
- 投票セクション直前に漫画セクション（3ページ画像＋モーダル＋キーボードナビ）
- 投票ボタンを背景画像カード（2×2グリッド、aspect-ratio:1、短ラベル18px白文字）に変更

### Step 5: 画像変換（自動 — 画像配置後）

ユーザーが画像を配置したら実行:
- 漫画ページ → WebP 100KB以下に圧縮
- 投票画像 → 240×240にリサイズ、WebP 20KB以下に圧縮
- `.webp.png` 等の二重拡張子があれば修正

## テーマ別スタンスマップ情報（キャラ設計の参考）

### 1. 高齢者免許返納（elderly-license）

**スタンスマップファイル:** `docs/elderly-license-stance-map.html`（117件）
**2軸定義:**
- X軸: stance_restriction（-2:強制不要 〜 +2:義務化賛成）
- Y軸: stance_priority（-2:移動権優先 〜 +2:交通安全優先）

**象限分布:**
| 象限 | 件数 | 割合 |
|------|------|------|
| 右上: 義務化賛成×安全優先 | 79件 | 68% |
| 左下: 強制反対×移動権優先 | 21件 | 18% |
| 左上: 強制反対×安全優先 | 2件 | 2% |
| 中立 | 15件 | 13% |

**感情強度:** 高感情(e≥1.5) 40件、冷静(e≤0) 29件

**代表投稿（台詞のヒント）:**
- Q1(義務化賛成・激怒): 「８５歳以上の免許返納義務化を強く要望」「義務返納と厳罰化を強く求める怒りの意見」
- Q3(強制反対・感情的): 「地方在住者が返納すれば生活不能と強く主張」「地方在住者が返納強制に強く反対し移動権最優先で感情的な意見」
- 冷静系: 「踏み間違いは年齢だけでなく誰にでも起き得るとし、技術導入や安全対策を重視する冷静な意見」「自主返納後の継続的代替交通の整備を指摘」

**推奨キャラ設計:**
- **キャラA（義務化賛成×安全優先、79件の声）**: 40代、都市部在住の会社員か主婦。高齢者の危険運転のニュースを見てきた。「制限をかけないと家族が危ない」という強い恐れと怒り。
- **キャラB（強制反対×移動権優先、21件の声）**: 地方在住の70代か、地方在住の高齢親を持つ50代。「車がなければ病院にも行けない」という切実な現実。強制に強く反発。

**対立の核心:** 都市の安全論 vs 地方の生存論。どちらも切実。

### 2. 自転車青切符（bike-blue-ticket）

**スタンスマップファイル:** `docs/bike-blue-ticket-stance-map.html`（116件）
**2軸定義:**
- X軸: stance_enforcement（-2:取締り不要 〜 +2:取締り強化賛成）
- Y軸: stance_priority（-2:自転車利用者の自由優先 〜 +2:歩行者・社会安全優先）

**象限分布:**
| 象限 | 件数 | 割合 |
|------|------|------|
| 右上: 取締賛成×安全優先 | 55件 | 47% |
| 左下: 取締反対×利用者優先 | 25件 | 22% |
| 左上: 取締反対×安全優先 | 13件 | 11% |
| 中立 | 22件 | 19% |

**感情強度:** 高感情(e≥1.5) 32件、冷静(e≤0) 35件

**代表投稿（台詞のヒント）:**
- Q1(取締賛成・怒り): 「自転車の一時不停止に怒り、厳しく取り締まるべきと主張」「自転車免許制度導入を強く求める」
- Q3(取締反対・感情的): 「インフラ整備なしの青切符制度に強く反対する署名」「自転車に強く反対し自由制限を懸念」

- **キャラA（取締賛成×安全優先、55件の声）**: 30〜40代、徒歩や車移動が多い。信号無視の自転車にヒヤリとした経験がある。「歩行者の命が危ない」。
- **キャラB（取締反対×利用者優先、25件の声）**: 20〜30代、自転車通勤者。「自転車道もないのに取り締まるのは理不尽」という怒り。

**対立の核心:** 同じ道路で交錯する「守られたい歩行者」と「走れる場所を奪われた自転車乗り」。

**対立の核心:** 「子どもの部活体験を守りたい親」 vs 「倒れるほど働く教員」。どちらも正しい。

### 4. 憲法改正（constitutional-amendment）

**スタンスマップファイル:** `docs/constitutional-amendment-stance-map.html`（366件）
**2軸定義:**
- X軸: stance_amendment（-2:改憲に強く反対・護憲 〜 +2:改憲に強く賛成）
- Y軸: stance_priority（-2:平和主義・9条護持を最優先 〜 +2:安全保障強化・現実的防衛を優先）

**象限分布:**
| 象限 | 件数 | 割合 |
|------|------|------|
| 左下: 改憲反対×平和主義優先 | 135件 | 37% |
| 右上: 改憲賛成×安保優先 | 67件 | 18% |
| 中立含む（X=0またはY=0） | 161件 | 44% |
| 左上: 改憲反対×安保優先 | 1件 | <1% |
| 右下: 改憲賛成×平和主義優先 | 2件 | 1% |

**感情強度:** 高感情(e≥1.5) 113件、冷静(e≤0) 116件（ほぼ拮抗）

**代表投稿（台詞のヒント）:**
- 左下(護憲・怒り): 「護憲・平和主義の立場から改憲に強く反対し感情的に批判」「軍拡を推進する高市政権下での改憲は絶対阻止」
- 右上(改憲賛成・激怒): 「改憲・自衛隊明記を熱烈支持、護憲派を激烈批判」「現行憲法は時代遅れ・現実に合わない」
- 冷静系: 「国民投票法の改正は改憲の地ならしという意見はあるだろうし、反対で意志を示したのはそれはそれで理解できる」

**推奨キャラ設計:**
- **キャラA（改憲反対×平和主義優先、135件の声）**: 60〜70代、戦後教育を受けた世代。「9条があったから戦後80年、日本は戦争しなかった。それを変えるなんて」という揺るぎない信念。
- **キャラB（改憲賛成×安保優先、67件の声）**: 30〜40代、国際情勢に敏感な世代。「北朝鮮のミサイル、中国の脅威、現実を見ろ。9条の美しい言葉で守れるのか」という焦り。

**対立の核心:** 「戦争の記憶から来る平和主義」 vs 「現実の脅威から来る安全保障論」。世代の分断でもある。
