HOW TO USE

3ステップで、話題の全体像へ

正解や多数派を決めるためではなく、「どんな考えが、どこにあるのか」を知るためのサイトです。

HOW TO READ

マップの色は「勝ち負け」ではありません

それぞれの色は、現在の考えに近いスタンスを示します。どの立場にも異なる理由や濃淡があります。

賛成・支持提案や方針を肯定的に評価
反対・批判問題点やリスクを重視
条件付き一定の条件や改善を求める
中立・保留情報不足や複数の視点を重視
※ 投稿数と分類比率は収集できたSNSサンプルの傾向です。社会全体の意見比率を表す世論調査ではありません。
FAQ

よくある質問

投票に会員登録は必要ですか？

必要ありません。公開中のテーマページから、そのまま投票できます。

SNSの分類結果は世論を表していますか？

いいえ。特定の期間と検索条件で収集できた投稿サンプルの傾向です。一般的な世論調査とは異なります。

AIの分類はすべて正確ですか？

AIによる論点とスタンスの分類には誤りが含まれる可能性があります。分類方法やデータの限界は「データについて」で公開しています。

投票した内容は変更できますか？

テーマページの投票画面に変更操作がある場合は、そこから現在の考えに合わせて更新できます。
